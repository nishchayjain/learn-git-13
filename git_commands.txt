git init -> Initialising a git repository
git clone -> download a repository from remote to local
git remote add origin -> adding the origin of remote to local
git status -> shows the current status of your git repository
git add -> adds the files to staging area
git commit -> commits the file to git versioning system
git push origin branch_name -> Push the code from local to remote in branch_name
git branch -> shows the current selected branch
git fetch -> fetches newly created code and branches on remote to local.
